<p>个人纯碎为了练习nodejs而做出来的一个博客，给个Star就是我最大的动力！前期页面样式写的比较简陋，有时间我在细心写</p>
<h4>技术选型：</h4>
<ul>
  <li>前端使用： &nbsp;ejs模板</li>

  <li> models&nbsp;&nbsp;&nbsp;封装了对数据库的接口</li> 
	<li>node_modules&nbsp;&nbsp;&nbsp;项目依赖包</li>
